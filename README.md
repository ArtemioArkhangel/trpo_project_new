Password Generator

Работа с проектом

Для сборки программы необходимо выполнить команду make all в корневой папке проекта. Исполняемый файл pwgen будет создан в каталоге bin. Запуск программы осуществляется командой ./bin/pwgen, после чего пользователь в интерактивном режиме задаёт длину пароля, количество и наборы используемых символов. Для запуска модульных тестов предназначена команда make test, для очистки скомпилированных файлов — make clean. Пайплайн непрерывной интеграции настроен через GitHub Actions и запускается автоматически при каждом пуше в основную ветку репозитория.
